# Vélo Hiver — graphes routiers pré-générés

Graphes du réseau routier (voies automobiles) des villes supportées par
l'app Vélo Hiver, utilisés pour l'aimantation des tracés d'itinéraires.

Données © les contributeurs d'[OpenStreetMap](https://www.openstreetmap.org/copyright),
sous licence [ODbL](https://opendatacommons.org/licenses/odbl/).
Générés via l'API Overpass, une fois par saison.

Format : `{ bboxes, nodes: [[id, lat, lon, intersection]], edges: [[a, b]] }`
